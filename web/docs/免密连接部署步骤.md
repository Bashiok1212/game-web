# 阿里云免密连接 - 部署步骤

使用阿里云 **Workbench 远程连接**，在浏览器里操作服务器，无需本地 SSH 和密钥。

---

## 第一步：通过浏览器连接 ECS

1. 登录 [阿里云 ECS 控制台](https://ecs.console.aliyun.com)
2. 找到你的实例，点击实例 ID 进入详情
3. 点击右上角 **远程连接**
4. 选择 **通过 Workbench 远程连接**
5. 点击 **立即登录**

> 首次可能提示设置 VNC 密码，按提示设置即可，之后可免密登录。

---

## 第二步：安装 Node.js

在 Workbench 终端依次执行：

```bash
apt update && apt upgrade -y
```

```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs
```

```bash
node -v
npm -v
```

看到 `v18.x.x` 即成功。

---

## 第三步：上传项目

### 方式 A：Workbench 文件管理（推荐）

1. 在 Workbench 窗口左侧或上方找 **文件** / **文件管理** 图标
2. 本地把 `h:\new-game\web` 打成 zip（选中 web 文件夹内所有内容 → 右键 → 发送到 → 压缩文件夹）
3. 在文件管理里上传 zip 到 `/root/`
4. 终端执行：

```bash
cd /root
unzip web.zip -d web
# 若 zip 内含 web 子文件夹，可能得到 /root/web/web，可执行：mv web/web/* web/ 后删除空目录
```

### 方式 B：云助手发送文件

1. ECS 控制台 → 实例详情 → **云助手** → **发送文件**
2. 选择本地的 `web` 文件夹打包的 zip
3. 目标路径填 `/root/`
4. 发送后在终端 `cd /root` 解压

### 方式 C：Git 克隆（若已推送到 GitHub/Gitee）

```bash
cd /root
git clone https://你的仓库地址.git
mv 仓库名 web  # 若需要
```

---

## 第四步：启动项目

在终端执行：

```bash
cd /root/web
npm install
export JWT_SECRET="改成你的随机密钥"
npm start
```

新开浏览器标签访问：`http://你的公网IP:3000`

---

## 第五步：PM2 保持运行

```bash
npm install -g pm2
cd /root/web
JWT_SECRET="你的密钥" pm2 start server.js --name game-web
pm2 save
pm2 startup
```

按提示执行 PM2 输出的命令。

---

## 安全组检查

若无法访问，在 ECS 控制台 → 安全组 → 添加入方向规则：端口 **3000**，授权对象 `0.0.0.0/0`。
