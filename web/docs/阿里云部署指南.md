# 阿里云部署指南 - 一步一步完成

从零开始，将游戏网站部署到阿里云。

---

## 第一步：注册阿里云账号

1. 打开 [阿里云官网](https://www.aliyun.com)
2. 点击右上角 **免费注册**
3. 按提示完成手机/邮箱验证
4. 实名认证（个人或企业，个人认证即可）

> 新用户通常有优惠，如轻量应用服务器首年约 60 元起

---

## 第二步：购买服务器

### 推荐：轻量应用服务器（适合入门）

1. 登录 [阿里云控制台](https://ecs.console.aliyun.com)
2. 左侧菜单找到 **轻量应用服务器**，或搜索「轻量」
3. 点击 **创建服务器**
4. 配置建议：
   - **地域**：选离你/玩家近的（如华东-杭州）
   - **镜像**：**Ubuntu 22.04**
   - **套餐**：最低档即可（约 2核1G，24元/月起）
   - **时长**：按需选择
5. 设置 **root 密码**（务必记住）
6. 提交订单并支付

### 备选：ECS 云服务器

若需要更多配置，可选 **云服务器 ECS**，流程类似，镜像同样选 Ubuntu 22.04。

---

## 第三步：配置安全组（放行端口）

1. 进入 **轻量应用服务器** 控制台
2. 点击你的服务器实例
3. 进入 **防火墙** 标签
4. 添加规则：
   - **端口**：`3000`（或你用的端口）
   - **协议**：TCP
   - **来源**：`0.0.0.0/0`（允许所有 IP）
5. 同时确保 **22** 端口已开放（SSH 登录用）

---

## 第四步：SSH 连接服务器

### Windows 用户

1. 打开 **PowerShell** 或 **命令提示符**
2. 使用服务器公网 IP 连接：

```bash
ssh root@你的服务器公网IP
```

3. 输入购买时设置的 root 密码

### 若提示「ssh 不是内部或外部命令」

- 安装 [Git for Windows](https://git-scm.com/download/win)，自带 SSH
- 或用 [PuTTY](https://www.putty.org/) 连接

---

## 第五步：在服务器上安装 Node.js

连接成功后，在终端依次执行：

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 验证
node -v   # 应显示 v18.x.x
npm -v
```

---

## 第六步：上传项目到服务器

### 方式 A：用 Git（推荐）

**本地：**

1. 在 `h:\new-game` 初始化 Git（若还没有）：
   ```bash
   cd h:\new-game
   git init
   git add .
   git commit -m "init"
   ```

2. 推送到 GitHub / Gitee 私有仓库

**服务器：**

```bash
# 安装 git
apt install -y git

# 克隆（替换为你的仓库地址）
git clone https://github.com/你的用户名/你的仓库.git
cd 你的仓库/web
```

### 方式 B：用 SCP 直接上传

在 **本地 PowerShell** 执行（替换 IP 和路径）：

```powershell
scp -r h:\new-game\web root@你的服务器IP:/root/
```

### 方式 C：用 FTP 工具

- 安装 [FileZilla](https://filezilla-project.org/) 或 [WinSCP](https://winscp.net/)
- 用 SFTP 连接服务器，上传 `web` 文件夹

---

## 第七步：在服务器上运行项目

```bash
cd /root/web   # 或你上传的路径

# 安装依赖
npm install

# 设置生产环境密钥（重要！）
export JWT_SECRET="随机生成一串长字符串"

# 直接运行（测试用）
npm start
```

浏览器访问：`http://你的服务器IP:3000`

若能看到页面，说明部署成功。

---

## 第八步：使用 PM2 保持进程常驻

```bash
# 安装 PM2
npm install -g pm2

# 启动并设为开机自启
cd /root/web
JWT_SECRET="你的密钥" pm2 start server.js --name game-web
pm2 save
pm2 startup
```

---

## 第九步：配置 Nginx 反向代理（可选，推荐）

使用 80 端口，无需在网址后加 `:3000`：

```bash
apt install -y nginx

# 编辑配置
nano /etc/nginx/sites-available/default
```

在 `server` 块内添加：

```nginx
location / {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
```

重启 Nginx：

```bash
nginx -t
systemctl restart nginx
```

防火墙放行 **80** 端口，即可用 `http://你的IP` 访问。

---

## 第十步：绑定域名（可选）

1. 在阿里云 **域名** 服务购买域名
2. 在域名解析中添加 **A 记录**，指向服务器公网 IP
3. 在 Nginx 中配置 `server_name 你的域名`，并启用 HTTPS（可选）

---

## 常见问题

| 问题 | 处理 |
|------|------|
| 无法访问 | 检查防火墙是否放行 3000/80 端口 |
| 502 Bad Gateway | 确认 Node 进程在运行（pm2 list） |
| 注册失败 | 检查 `data` 目录权限，确保可写 |
